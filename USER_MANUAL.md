## 1. 快速入门

### 1.1 访问地址

```
https://www.4si.cc/PMO/
```

### 1.2 首次使用

1. 打开地址，如系统未初始化，自动进入**管理员初始化向导**
2. 设置管理员用户名和密码（≥6位）
3. 初始化完成后自动跳转登录页
4. 管理员登录后进入管理后台，可创建其他用户账号
5. 普通用户由管理员分配账号后登录，直接进入看板

### 1.3 界面总览

```
┌─────────────────────────────────────────────┐
│ 📋 项目管理看板   👤 用户  📊 项目配额  💾 KB配额 │  ← 顶栏
│ [＋新建] [重置演示] [甘特图] [导出] [改密] [退出] │
├──────────┬──────────────────────────────────┤
│ 当前项目  │  阶段1 │ 阶段2 │ 阶段3 │ ...      │  ← 阶段区
│  ├ 项目A  │──────────────────────────────────│
│  ├ 项目B  │  任务表格（名称/工期/状态/依赖…）  │  ← 任务区
│  └ 项目C  │                                  │
│          │                                  │
│ 知识库    │                                  │
├──────────┴──────────────────────────────────┤
```

## 2. 登录与账号

### 2.1 登录

1. 打开登录页，输入**用户名**和**密码**
2. 点击「登录」
3. 管理员自动跳转管理后台，普通用户进入看板

### 2.2 账号锁定

- 连续 **5 次**密码错误 → 账号锁定 **15 分钟**
- 锁定期间登录页会显示剩余锁定时间
- 锁定到期后自动解除

### 2.3 会话管理

- 登录有效期：**24 小时**（可后台调整）
- 关闭浏览器标签即退出
- 主动退出：点击右上角「退出」

## 3. 项目管理

### 3.1 项目列表

左侧边栏显示当前账号的所有项目。点击项目名称切换。

### 3.2 新建项目

1. 点击「＋ 新建项目」
2. 输入项目名称和描述
3. 系统自动创建默认阶段和模板任务
4. 项目数量受配额限制（默认 10 个）

### 3.3 删除项目

1. 切换到目标项目
2. 点击项目名旁的删除图标
3. 确认删除（**不可恢复**）

### 3.4 重置演示数据

点击「🔄 重置演示数据」可将当前项目恢复到初始模板状态（含 29 条示例任务）。

## 4. 任务看板

### 4.1 任务层级

任务支持 4 级树形结构：

```
📁 1. 需求分析          ← 一级任务
  ├ 1.1 用户访谈        ← 二级子任务
  │  ├ 1.1.1 访谈记录   ← 三级子任务
  │  └ 1.1.2 需求整理
  └ 1.2 竞品分析
```

- 点击任务行前的 `▶` 展开/收起子任务
- 自动编号按层级重编

### 4.2 创建任务

1. 点击底部「＋ 新建任务」或在表格最后一行直接输入
2. 填写任务信息（见下方字段说明）

### 4.3 编辑任务

- **双击**任务行任意单元格进入编辑模式
- 修改后点击其他位置自动保存

### 4.4 任务字段说明

| 字段 | 说明 | 必填 | 示例 |
|------|------|------|------|
| 任务名称 | 任务描述 | ✅ | "数据库设计" |
| 工期(天) | 预计工作天数 | ✅ | 5 |
| 计划开始 | YYYY-MM-DD | — | 2026-07-01 |
| 计划结束 | YYYY-MM-DD | — | 2026-07-05 |
| 实际结束 | YYYY-MM-DD | — | 2026-07-06 |
| 状态 | 未开始/进行中/已完成/暂停 | ✅ | 进行中 |
| 前置依赖 | 依赖的任务 ID（逗号分隔） | — | "id_1001,id_1002" |
| 备注 | 文字备注 | — | "需协调A部门" |

### 4.5 批量操作

点击「批量填充」可对选中任务批量设置：
- 状态
- 计划开始/结束日期

### 4.6 前置依赖

- 在「前置依赖」列输入依赖任务的 ID
- 多个依赖用**英文逗号**分隔
- 计划开始时间会自动延后到所有前置任务完成之后

## 5. 甘特图

### 5.1 切换视图

点击「📊 甘特图」按钮或按 `Ctrl+G` 切换。

### 5.2 甘特图图例

| 颜色 | 含义 |
|------|------|
| 🟢 绿色 | 已完成 |
| 🔵 蓝色 | 进行中 |
| 🔴 红色 | 已延期（计划结束 < 今天且未完成） |
| ⚪ 灰色 | 未开始 |

### 5.3 阅读方法

- 每行 = 一个任务
- 横轴 = 日期（按天展开）
- 蓝色块 = 任务持续时间
- 鼠标悬停查看详情

## 6. 知识库

### 6.1 功能说明

知识库用于存储项目相关文件和链接，按分类管理。

### 6.2 上传文件

1. 点击「📚 知识库」
2. 选择分类（或新建分类）
3. 点击「上传文件」选择本地文件
4. 单个文件 ≤ 50MB，总容量 ≤ 100MB/账号

### 6.3 添加链接

1. 点击「添加链接」
2. 输入链接名称和 URL
3. 保存

### 6.4 配额

- 右上方实时显示已用/总量
- 超配额时上传会被拒绝
- 管理员可在后台调整

## 7. 导出报表

### 7.1 导出 Excel

点击「📥 导出报表」或按 `Ctrl+S`。

### 7.2 报表内容

导出的 Excel 包含 **2 个 Sheet**：

| Sheet | 内容 |
|-------|------|
| 任务清单 | 完整任务表（含层级、状态、工期、依赖、备注） |
| 甘特图 | 按月日着色的甘特图（已完成🟢 / 进行中🔵 / 延期🔴 / 未开始⚪） |

### 7.3 注意事项

- 导出依赖 SheetJS 库，需网络加载（CDN）
- 离线环境需本地化 `xlsx.min.js`

## 8. 修改密码

1. 点击右上角「🔑 修改密码」
2. 输入当前密码
3. 输入新密码（≥6位，不能与当前密码相同）
4. 再次输入新密码确认
5. 点击「更新密码」

## 9. 常见问题

### Q: 页面显示"加载中..."不动了？
**A:** 刷新页面，如仍不行，清除浏览器 IndexedDB 数据后重试。

### Q: 为什么不能创建项目了？
**A:** 项目数已达配额上限（默认 10 个）。联系管理员调整配额。

### Q: 知识库上传失败？
**A:** 检查：① 单个文件 ≤50MB ② 总容量未超配额 ③ 浏览器支持 IndexedDB。

### Q: 密码忘了怎么办？
**A:** 联系管理员重置密码（管理员在后台 → 账号管理 → 重置密码）。

### Q: 甘特图不显示？
**A:** 确保任务已设置「计划开始」和「计划结束」日期。

### Q: 导出的 Excel 没有甘特图？
**A:** SheetJS 库未加载成功，检查网络或刷新后重试。

### Q: 被锁定 15 分钟怎么办？
**A:** 等待倒计时结束，或联系管理员在后台重置密码后重新登录。
